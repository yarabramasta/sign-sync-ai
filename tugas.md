### 1. **Struktur Fungsional**

**Contoh**: Perusahaan Manufaktur

**Bagan:**

```
CEO
 |
 +-- Produksi
 |     +-- Manager Produksi
 |     +-- Supervisor Produksi
 |
 +-- Pemasaran
 |     +-- Manager Pemasaran
 |     +-- Tim Pemasaran
 |
 +-- Keuangan
 |     +-- Manager Keuangan
 |     +-- Tim Akuntansi
 |
 +-- Sumber Daya Manusia
       +-- Manager SDM
       +-- Tim Rekrutmen
```

### 2. **Struktur Divisional**

**Contoh**: Perusahaan Multinasional

**Bagan:**

```
CEO
 |
 +-- Divisi Amerika Utara
 |     +-- Manager Regional
 |     +-- Tim Fungsional (Produksi, Pemasaran, Keuangan)
 |
 +-- Divisi Eropa
 |     +-- Manager Regional
 |     +-- Tim Fungsional (Produksi, Pemasaran, Keuangan)
 |
 +-- Divisi Asia
       +-- Manager Regional
       +-- Tim Fungsional (Produksi, Pemasaran, Keuangan)
```
